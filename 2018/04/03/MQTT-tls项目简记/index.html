MQTT-tls项目简记

项目设计目的

终端采用开放物联网MQTT-tls协议，实现安全认证和传输、订阅和发布。投标参与方应提交MQTT通信相关的参数文件，并提前完成配置。参数包括：

  1. MQTT 代理broker地址和端口
  2. 设备编号
  3. Tls x509证书
  4. 设备数据发布topic
  5. 设备log发布topic
  6. 设备订阅topic

###环境搭建
1.PC端需要安装Paho MQTT C client library库，由于SSL不包含在库里，需要单独编译OpenSSL并添加进工程。
注意在https://www.openssl.org/source/官方网站下载1.0.2o.tar.gz文件，否则ms/中没有do_nasm.bat文件。
2.下载OpenSSL源码，Perl，NASM，运行时提示dmake未安装，ppm install dmake安装中。
打开VS2013 X64本机工具命令提示
运行D:\Microsoft Visual Studio 12.0\VC\bin\amd64\vcvars64.bat
切到openssl目录下
运行perl Configure VC-WIN64A
运行ms\do_nasm
运行ms\do_win64a
运行nmake -f ms\nt.mak
执行、测试：nmake -f ms\nt.mak test
检查上一步编译是否成功，如果成功则最后显示“passed all tests”字样
nmake -f ms\nt.mak install
成功则会在打印的输出信息里显示了目录路径里生成bin、include、lib、ssl四个文件夹
参考链接https://geekori.com/details.php?que_id=17199
